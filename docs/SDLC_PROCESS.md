# SDLC Process

This project follows Agile SDLC with DevSecOps controls.

## Stage Gates

1. BRD review complete.
2. Epics and user stories approved.
3. Test cases and traceability approved.
4. Design approved.
5. Code and test scripts implemented.
6. CI, security, and deployment checks passed.
7. Release readiness approved.

## Traceability

Every user story, test case, design decision, and implementation item should trace back to the BRD or an approved clarification.

## Delivery Mode

Work incrementally. Each stage should produce reviewable artefacts before moving to the next stage.
