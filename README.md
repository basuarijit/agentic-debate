# Agentic Debate

Scaffold repository for an AI-agent debate application built with React, Python, FastAPI, Uvicorn, OpenAI, LangChain, and LangGraph.

This repository is intentionally scaffold-only at this stage. SDLC artefacts and application code will be created step by step from the BRD.

## Current Scope

- Project guidance for Codex-driven development.
- Backend and frontend directory boundaries.
- Placeholder folders for Agile SDLC artefacts.
- DevSecOps governance placeholders.

## Planned SDLC Sequence

1. Create epics and user stories from the BRD.
2. Create test cases and traceability.
3. Create solution design.
4. Implement backend and frontend code with tests.
5. Add CI/CD, security checks, and deployment automation.
